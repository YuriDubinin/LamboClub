The project is an introductory catalog of sports cars, the content for which is taken from the Internet for clarity

The project structure is built using the Flex module. Unobtrusive animation was applied to liven up the layout. Simple transitions are used for the same purpose. A slight bias is made on adaptive video and images.

The bias in the creation of the HTML structure was made for compliance with the BAM methodology. The project uses the SASS preprocessor in SCSS syntax.

Also, the bias is made on media queries, which allows you to display content on devices with different extensions. JavaScript is used to a minimum, in solutions related to header and footer.
